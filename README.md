# Mind the Gender Gap: Bibliometrics of Economic Research on Gender Wage Gap
This project plans to apply the bibliometric method to map the landscape of the gender wage gap literature using the Scopus database. 

`data_preparation.py` is for data collection with api obtained from: https://dev.elsevier.com/ and data preprocessing

`bibliometrics_analysis.ipynb` is for applying descriptive analysis and network methods to uncover the key contributors and the structure and dynamics of constituents in this field.
